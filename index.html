01리디북스

Open API , Json

Point Color

국내 웹 도서 사이트 리디북스의 클론 사이트입니다.
페이지 내의 도서들을 Kakao open Api를 사용하여 구현하였습니다.

도서 사이트의 특징인 많은 정보로 인하여 복잡한 레이아웃을 좀 더
간결하게 표현하고 많은 정보를 보여주기 위해 슬라이더를 넣어
사용자가 좀 더 다양한 정보를 볼 수 있게 제작하였습니다.

jQuery를 활용해 슬라이더를 넣어 동적인 느낌을 주고
상품 리스트, 게시판 등의 다양한 인터렉션 요소를 배치하였습니다.

02로마네

Shopping Website

Point Color

정보를 제공되게끔 제작하였습니다

03Mokko

Responsive Website

Point Color

해외 웹 레스토랑 사이트 MOKKO의 클론 사이트입니다.
반응현 사이트로 여러 기종에도 깔끔하게 볼 수 있도록 media로
제작하여 호환성이 좋은 사이트입니다.

음식의 다채로운 느낌을 표현하기 위해 애니메이션을 사용하여
모션을 넣고 사이트를 방문하는 방문자에게 보는 즐거움을 선사하였습니다.

레스토랑의 음식을 돋보이게 레이아웃은 간결하게 제작하고
서브페이지를 제작하여 메인페이지에 링크를 연결하였습니다.

04ROYAL ACADEMY

Responsive Website

Point Color

해외 웹 ROYAL ACADEMY OF MUSIC의 클론 사이트입니다.
그동안 제작했던 사이트들과 차별점을 두어 css로 다채로운
레이아웃을 연출하였습니다.

아카데미 홍보용 사이트라는 특징을 브로슈어 느낌으로 제작하여
아카데미만의 니즈를 표하였습니다.

애니메이션을 사용하여 게시판을 율동감있게 표현하고
강렬한 레드와 블랙색상으로 포인트를 주어
사이트를 방문하는 방문자에게 보는 즐거움을 주었습니다.
